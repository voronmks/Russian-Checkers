♟️ Russian Checkers – Multi‑Language Edition
A fully featured Russian Checkers (also known as Shashki) game for the terminal.
Play against a friend or challenge a simple AI opponent. Enjoy classic rules: diagonal movement, mandatory captures, king promotion, and multi‑jump sequences.
Built in 7 programming languages – each implementation is self‑contained, with a clean interface and advanced features.

✨ Features
8×8 board with standard starting positions.

Mandatory capture – you must take a piece if possible (multi‑jump support).

King promotion – a piece reaching the last row becomes a king (can move and capture backwards).

Simple AI – random legal moves, or a basic minimax with depth 1–2 for demo.

Highlighting – possible moves are shown with visual feedback.

Undo move – press u to undo (optional, implemented in most languages).

Save/Load – save the current game state to a file and load it later (optional).

Cross‑platform – runs in any terminal (Windows, macOS, Linux).

🚀 How to Run
Each file is standalone – run it with the appropriate interpreter/compiler.

Language	Command
Python	python checkers.py
Go	go run checkers.go
JavaScript	node checkers.js
C#	dotnet run (or csc Checkers.cs && Checkers.exe)
Java	javac Checkers.java && java Checkers
Ruby	ruby checkers.rb
Swift	swift checkers.swift
🎮 Controls
Enter move – type coordinates like a3-b4 (from‑to).

Multi‑jump – enter each step sequentially (e.g., a3-b4 then b4-c5).

Undo – press u to undo the last move.

Save/Load – press s to save, l to load (where implemented).

Quit – press q to exit.
